Understanding Web Architecture

Web architecture is the basic structure of how a website works. It shows how the client (browser), server, and internet work together. When you open a website, the client sends a request, the server processes it, and then sends the webpage back to be shown on your screen.

Bonus: Why we need a server

We need a server to show websites on the internet because local files on our computer cannot be accessed by other people worldwide. A server is always online and has a public address, allowing anyone with the link to request and view the website files.
